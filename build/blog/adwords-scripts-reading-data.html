Google Ads Scripts: Reading Data

written on 2018-03-22

How to read data from Google Ads scripts?

In the Brief Introduction to Google Ads Scripts, we saw how to create our first Google Ads script. Now we‚Äôll try to do something useful with it. One of the first steps is to learn how to read campaigns data from Google Ads scripts.

Getting a list of running campaigns

To get a list of campaigns, we‚Äôll use AdsApp.CampaignSelector. As a very basic first example, we‚Äôll try to fetch all of our running campaigns.

First of all, to structure our code a bit, we‚Äôll create a function outside of main(). We can call it getAllActiveCampaigns() and declare it that way: function getAllActiveCampaigns() {}. Remember that only what‚Äôs inside main() will run, so we‚Äôll have to call getAllActiveCampaigns() inside it later but we‚Äôll omit it for now.

AdsApp.campaigns().get() is going to fetch all campaigns. If we want to be able to re-use it, we have to store it in a variable:

function getAllActiveCampaigns() {
  var campaigns = AdsApp.campaigns().get();
}

But remember that we only want running campaigns. We can use .withCondition() to introduce filtering:

function getAllActiveCampaigns() {
  var campaigns = AdsApp
      .campaigns()
      .withCondition('Status = ENABLED')
      .get();
}

Note the line indentation introduced for more readability. It is 100% optional but very recommended.

Other examples of conditions may be: .withCondition('Impressions > 1000'), .withCondition("Name CONTAINS 'Canada'")‚Ä¶ Those conditions are also chainable.

In order to help us, Google Ads is returning a few helpers alongside the list of campaigns we fetched. It can be really helpful to loop over the results. Here is an example with a while() {} loop, along with Google Ads scripts‚Äô .next() and .hasNext() helpers:

function getAllActiveCampaigns() {
  var campaigns = AdsApp
      .campaigns()
      .withCondition('Status = ENABLED')
      .get();

  while (campaigns.hasNext()) {
    var campaign = campaigns.next();
  }
}

Now if you try to run this, it wouldn‚Äôt do anything, because the loop is basically empty. Let‚Äôs log the name of the campaign: Logger.log(campaign.getName());.

Let‚Äôs try to do a bit more. Campaign budget can be easily fetched with .getBudget() but it only returns an integer (for example, for a budget of $100, the response will be 100), so let‚Äôs try to get the currency code too. AdsApp.currentAccount().getCurrencyCode() should do!

Here is the final result:

function getAllActiveCampaigns() {
  var campaigns = AdsApp
      .campaigns()
      .withCondition('Status = ENABLED')
      .get();

  while (campaigns.hasNext()) {
    var campaign = campaigns.next();
    Logger.log(campaign.getName());
    Logger.log('Daily Budget: ' + campaign.getBudget() + AdsApp.currentAccount().getCurrencyCode());
  }
}

function main() {
  getAllActiveCampaigns();
}

Getting a list of ad groups

From what we‚Äôve just learnt with how to read campaigns above, getting a list of ad groups will be really easy. We only need to switch AdsApp.campaigns() with AdsApp.adGroups() as well as some variables‚Äô names.

function getAllAdGroups() {
  var adGroups = AdsApp
      .adGroups()
      .get();

  while (adGroups.hasNext()) {
    var adGroup = adGroups.next();
    Logger.log(adGroup.getName());
  }
}

As budgets are placed on a campaign level, you would get an error trying to run adGroup.getBudget() so I simply ommited it here.

Getting a list of ads inside an ad group

Same thing with ads, we can simply replace AdsApp.adGroups() by AdsApp.ads(). But let‚Äôs do things a bit differently this time. Let‚Äôs get the ads inside a specific ad group.

We can simply select an ad group by its name with .withCondition(). if (adGroup.hasNext()) {} condition is present in case our ‚Äúsearch‚Äù didn‚Äôt result any ad group. After that, we can log the part 1 & part 2 headlines (in the case of Expanded text ads).

function getAdsInAdGroup() {
  var adGroup = AdsApp
      .adGroups()
      .withCondition("Name = 'Ad Group #1'")
      .get();

  if (adGroup.hasNext()) {
    var ads = adGroup.next().ads().get();
    while (ads.hasNext()) {
      var ad = ads.next();
      Logger.log(ad.getHeadlinePart1());
      Logger.log(ad.getHeadlinePart2());
    }
  }
}

You should now be comfortable reading campaigns, ad groups & ads from Google Ads scripts!
